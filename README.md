# Fake News Detection using Machine Learning

## Overview
This project detects fake news using NLP and machine learning techniques.

## Features
- 4-stage feature engineering (BoW, TF-IDF, Domain Features)
- Multiple ML models (Logistic Regression, SVM, Random Forest, XGBoost)
- Ablation study
- ROC curve and evaluation metrics

## Dataset
ISOT Fake News Dataset

## Results
Achieved ~99% accuracy using TF-IDF + domain features.
